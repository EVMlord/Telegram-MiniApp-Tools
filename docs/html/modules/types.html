Module types

The types module defines all TypeScript types and interfaces used across the Telegram MiniApp project. These type definitions ensure type safety and clarity when interacting with the Telegram Web App API and MiniApp data.

Key Definitions:

  • init-data.js: Initialization data for the Telegram Web App.
  • launch-params.js: Launch parameters for the MiniApp.
  • theme-params.js: Theme settings for the Telegram Web App.
  • telegram-web-app.js: API objects for the Telegram Web App.
