Interface FullscreenManager

Manages the fullscreen state and provides methods to control it.

interface FullscreenManager {
    destroy: () => void;
    exitFullscreen: () => void;
    getError: () => null | string;
    getIsFullscreen: () => boolean;
    offError: (callback: (error: null | string) => void) => void;
    offFullscreenChange: (callback: (isFullscreen: boolean) => void) => void;
    onError: (callback: (error: null | string) => void) => void;
    onFullscreenChange: (callback: (isFullscreen: boolean) => void) => void;
    requestFullscreen: () => void;
}

Properties

destroy

destroy: () => void

Cleans up event listeners and internal state. Should be called when the manager is no longer needed to prevent memory leaks.

exitFullscreen

exitFullscreen: () => void

Requests the app to exit fullscreen mode. If the request fails, the error can be retrieved using getError.
