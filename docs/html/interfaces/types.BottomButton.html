Interface BottomButton

This object controls the main button, which is displayed at the bottom of the Web App in the Telegram interface.

interface BottomButton {
    color: string;
    hasShineEffect: boolean;
    isActive: boolean;
    isProgressVisible: boolean;
    isVisible: boolean;
    position?: "left" | "top" | "bottom" | "right";
    text: string;
    textColor: string;
    disable(): BottomButton;
    enable(): BottomButton;
    hide(): BottomButton;
    hideProgress(): BottomButton;
    offClick(callback: () => void): BottomButton;
    onClick(callback: () => void): BottomButton;
    setParams(params: MainButtonParams): BottomButton;
    setText(text: string): BottomButton;
    show(): BottomButton;
    showProgress(leaveActive?: boolean): BottomButton;
}

Properties

color

color: string

Current button color. Set to themeParams.button_color by default.

hasShineEffect

hasShineEffect: boolean

Shows whether the button has a shine effect. Set to false by default.

isActive

isActive: boolean

Shows whether the button is active. Set to true by default.

isProgressVisible

isProgressVisible: boolean

Readonly. Shows whether the button is displaying a loading indicator.

isVisible

isVisible: boolean

Shows whether the button is visible. Set to false by default.

Optionalposition

position?: "left" | "top" | "bottom" | "right"

Position of the secondary button. Not defined for the main button. It applies only if both the main and secondary buttons are visible. Set to left by default. Supported values:

  • left, displayed to the left of the main button,
  • right, displayed to the right of the main button,
  • top, displayed above the main button,
  • bottom, displayed below the main button.

text

text: string

Current button text. Set to CONTINUE by default.

textColor

textColor: string

Current button text color. Set to themeParams.button_text_color by default.

    • is_visible - show the button.

  • A method to make the button visible. Note that opening the Web App from the attachment menu hides the main button until the user interacts with the Web App interface.

    Returns BottomButton

showProgress

  • showProgress(leaveActive?: boolean): BottomButton

  • A method to show a loading indicator on the button. It is recommended to display loading progress if the action tied to the button may take a long time. By default, the button is disabled while the action is in progress. If the parameter leaveActive=true is passed, the button remains enabled.
