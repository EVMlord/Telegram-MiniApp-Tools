Interface CloudStorage

This object controls the cloud storage. Each bot can store up to 1024 items per user in the cloud storage.

interface CloudStorage {
    getItem(key: string, callback?: CloudStorageGetItemCallback): void;
    getItems(keys: string[], callback?: CloudStorageGetItemsCallback): void;
    getKeys(callback?: CloudStorageGetKeysCallback): void;
    removeItem(key: string, callback?: CloudStorageRemoveItemCallback): void;
    removeItems(
        keys: string[],
        callback?: CloudStorageRemoveItemsCallback,
    ): void;
    setItem(
        key: string,
        value: string,
        callback?: CloudStorageSetItemCallback,
    ): void;
}

getItem

  • getItem(key: string, callback?: CloudStorageGetItemCallback): void

  • A method that receives a value from the cloud storage using the specified key.

    Parameters

    • key: string

      The key should contain 1-128 characters, only A-Z, a-z, 0-9, _ and - are allowed.

    • Optionalcallback: CloudStorageGetItemCallback

      In case of an error, the callback function will be called and the first argument will contain the error. In case of success, the first argument will be null and the value will be passed as the second argument.

    Returns void

setItem

  • setItem(
        key: string,
        value: string,
        callback?: CloudStorageSetItemCallback,
    ): void

  • A method that stores a value in the cloud storage using the specified key.

    Parameters

    • key: string

      The key should contain 1-128 characters, only A-Z, a-z, 0-9, _ and - are allowed.

    • value: string

      The value should contain 0-4096 characters. You can store up to 1024 keys in the cloud storage.

    • Optionalcallback: CloudStorageSetItemCallback

      If an optional callback parameter was passed, the callback function will be called. In case of an error, the first argument will contain the error. In case of success, the first argument will be null and the second argument will be a boolean indicating whether the value was stored.

    Returns void
